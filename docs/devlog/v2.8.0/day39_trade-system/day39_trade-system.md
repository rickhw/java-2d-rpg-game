# Video Summary

### 🗺️ 遊戲轉場效果
- **新增轉場效果**：為了改善
地圖切換的感覺，遊戲增加了視覺上的轉場效果。
- **更新遊戲狀態**：在切換地圖時，暫時將遊戲狀態設置為轉場狀態，並逐步更改螢幕透明度至幾乎完全黑暗。

### 👤 NPC 商人功能
- **實現商人角色**：引入一個可以買賣物品的商人NPC，並設計其對話和交易介面。
- **商人存貨系統**：商人現在擁有自己的物品清單以供玩家購買，並具備交易邏輯。

### 💵 買賣系統
- **基礎交易邏輯**：新增買賣物品功能，允許玩家檢視並選擇商人的商品進行購買。
- **價格設定**：每個物品都有設定價格，玩家必須擁有足夠的金幣才能成功購買物品。

### ⚔️ 物品管理
- **角色與商人的物品存放**：玩家和商人有各自的物品列表，允許撤回與存放物品進行管理。
- **裝備限制**：防止賣出已裝備的物品，增強遊戲的策略性。

### 🔄 使用者介面改進
- **對話與選項顯示**：商人的對話與選項介面經過優化，確保玩家操作直觀明瞭。
- **顯示玩家貨幣**：在交易介面中增加顯示玩家現有金幣的功能，提升用戶體驗。

### 🎮 最終測試與優化
- **測試交易系統**：進行了一系列測試，確認買賣功能正常運作，並確保所有交互能否順利進行。
- **鼓勵玩家探索**：增加商人交易功能後，玩家更有動力通過擊敗怪物獲得金幣以購購入各種設備和物品。
