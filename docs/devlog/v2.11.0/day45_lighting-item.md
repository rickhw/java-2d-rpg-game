# Video Summary

### 💡 亮度功能實現
- **亮度效果**: 增加燈籠裝備導入的亮度效果，周圍環境在沒有光照物品時會變得完全黑暗。
- **燈籠設計**: 複製燈籠圖像，並將其設計為可以像武器或盾牌一樣裝備或卸下的物品。

### 🛠️ 程式碼優化
- **去除多餘類別**: 刪除了創建不必要的區域類別，通過簡化代碼來實現相同的效果。
- **參數設置**: 添加了 "light radius" 屬性來區分不同光源的光照範圍，以便在未來可以添加多種光源。

### 📦 使用者介面調整
- **裝備系統擴展**: 擴展裝備類型，將燈籠作為新類型的物品，並更新介面以顯示其狀態。
- **燈光狀態更新**: 當玩家裝備或卸下燈籠時，會更新燈光狀態，以調整周圍的亮度。

### 🔄 優化性能
- **布林標誌**: 使用 `light updated` 布林值控制光源更新的頻率，以避免每幀重複調用更新方法，提高性能。
- **畫面渲染**: 根據玩家是否有裝備光源，調整畫面顏色及顯示方式，確保視覺效果良好。 

### 🌙 未来发展
- **新增功能展望**: 下一個計劃包括實現日夜循環系統，以增加遊戲的真實感和互動性。
