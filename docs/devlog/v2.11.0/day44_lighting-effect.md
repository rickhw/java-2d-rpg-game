# Video Summary

### 🌌 環境管理

- **環境管理器**: 創建一個名為 **EnvironmentManager** 的類，負責處理各種環境濾鏡，如光照、雨和霧等。

### 🖊️ 黑暗和光效

- **黑暗濾鏡**: 使用 **BufferedImage** 和 **Graphics2D** 來實現黑暗和光源區域。黑色矩形作為黑暗濾鏡，可以調整透明度來控制可見性。
- **光源區域**: 當有光源（如火把或燈籠）時，創建一個在玩家周圍的圓形光源區域。

### 🎨 漸層效果

- **漸層設計**: 使用 **Color** 和 **float** 陣列來創建中心最亮、邊緣逐漸變暗的漸層光效。調整陣列大小可以控制漸層的細節。
  
### ⚙️ 實現步驟

- **代碼步驟**: 通過代碼執行以下步驟，創建矩形和圓形區域，然後從矩形中減去圓形，最終在黑暗濾鏡中繪制出這一效果。

### 🔄 可調參數

- **參數調整**: 可以根據光源的類型（火把、燈籠等）調整圓形大小。最大尺寸不應超過螢幕寬高，以避免顯示問題。

### 🏰 地牢設計

- **應用效果**: 利用這一光影系統可以創建更生動的地牢級別，增添遊戲的氛圍。
