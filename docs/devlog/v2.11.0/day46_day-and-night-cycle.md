# Day and Night Cycle

- day state
- Tent (帳篷): 睡覺，然後不管狀態在哪個時間，都會回到 Day
    use case: inn, bed, home ... etc

---

# Video Summary

### 🌄 日夜循環實現
- **日夜循環**已經實作到遊戲中，包含四個階段：**白天**、**黃昏**、**夜晚**和**黎明**。
- 每個狀態持續的長度可自定義（例如，600幀代表10秒的白天）。

### 🌑 黑暗過渡效果
- 當狀態為黃昏時，利用**alpha值**來調整黑暗過濾器的透明度，使畫面逐漸變暗。
- 在夜晚狀態，**alpha值**會持續增加至1，使畫面完全黑暗。

### 🌞 明亮過渡效果
- 當狀態轉為黎明，**alpha值**開始減少，讓畫面逐漸變亮，直到回到白天狀態。

### 🏕️ 休息道具的設計
- 設計了一個**帳篷道具**，可以讓玩家跳過到下一個白天，並設計相應的使用效果。
- **帳篷道具**使用後會觸發畫面的漸變效果，增加遊戲的視覺體驗。

### 🎶 音效整合
- 為帳篷道具增加了一個**睡眠音效**，提升使用該道具時的沉浸感。
- 可以選擇讓該道具使用後消失，或是可無限使用以便於遊戲設計。

### 🛠️ 調整遊戲參數
- 玩家可以根據需要調整各個參數，例如白天持續的幀數和過渡速度，以最佳化遊戲體驗。
- 建議玩家根據自身遊戲的需求，將數值設定調整為合適的長度。 

### 🎮 自定義擴展
- 玩家可以進一步擴展遊戲，如設計自己的房子、床等，進一步加強遊戲中的休息系統。
- 鼓勵玩家在此基礎上加入個人創意，提升遊戲的趣味性和玩法多樣性。
