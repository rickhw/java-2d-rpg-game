# Options Menu

    - Music Volume
    - Sound Effect Volume

# Video Summary

### 🎮 遊戲選項菜單開發概述
- **全螢幕模式**：實現了開啟和關閉全螢幕功能，用戶可按 Escape 鍵進入選項菜單。
- **音樂和音效音量調整**：提供音樂和音效音量的調整選項，音量範圍設定在 -80 到 6 之間。

### ⌨️ 控制選項
- **鍵盤操作設定**：用戶可以檢查遊戲的鍵位設定，並可隨時返回選項菜單。
- **操作簡化**：增加返回選項以便輕鬆關閉選項菜單，按下 Enter 鍵可返回遊玩狀態。

### 📁 設定儲存
- **配置文件**：玩家的選項設定將被儲存至配置文件中，以便重新啟動遊戲時自動恢復這些設定。
- **重啟提示**：如果更改全螢幕選項，系統將提示用戶重啟遊戲以生效。

### 🎼 用戶界面設計
- **選項菜單佈局**：選項菜單以子窗口形式顯示，選項文字居中，讓用戶操作直觀易懂。
- **滑動條設計**：音量滑動條設置在選項菜單內，便於即時調整音量。

### 🔄 狀態管理
- **狀態切換**：利用狀態機進行管理，按下特定鍵可在不同子狀態間切換。
- **條件邊界檢查**：對於命令號進行邊界檢查，避免數值越界的情況影響功能。
