# Video Summary

### 🎮 遊戲更新概況
- 本次影片中，開發者展示了遊戲的**探索機制**更新，專注於可交互物體的設計，如**門**和**寶箱**。

### 🗝️ 互動與物件管理
- 向遊戲中新增了**障礙物體**類型，允許玩家在靠近這些物體時進行互動。
- 引入了**對話狀態**來提示玩家，例如當站在門前時提示「你需要一把鑰匙來開門」。

### 🔑 鑰匙使用功能
- 在玩家物品中添加了**鑰匙**物件，若玩家在門旁邊使用鑰匙，則可以開啟門。
- 增加了檢查來確保只有在門附近時，玩家使用鑰匙才會有效。

### 🥠 寶箱與物品獲取
- 寶箱新開發，玩家可透過**按鍵互動**檢查並開啟，獲得隨機物品。
- 寶箱開啟後會根據玩家的**背包狀態**提供不同的訊息，如背包已滿時提示「你無法再攜帶更多物品」。

### 📦 物品使用邏輯
- 當使用物品或鑰匙後，進行邏輯檢查來確保只有在合適情況下才能消耗物品，避免不必要的損失。
- 對於未能成功使用鑰匙的情況，系統會顯示相應的提示，並保持物品不變。

### 📅 下一步計劃
- 開發者提到接下來會增加**地下城**的設計，這一部分將在後續影片中詳細介紹。
