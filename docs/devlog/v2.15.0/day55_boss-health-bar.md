# Summary


### 🎮 遊戲設計更新

- **新增切換場景管理器**：設計了一個切換場景的管理器，以便在玩家遇到特定事件時新增過場動畫。
- **特殊血條顯示**：為最終boss戰設計了較大的血條，以增強戰鬥的戲劇感，並在血條上方顯示怪物名稱。

### 🔥 攻擊計算調整

- **火球攻擊力調整**：將火球的攻擊力從 **2** 降低到 **1**，引入了基於等級的計算系統，根據玩家等級提升攻擊強度。

### 📊 介面優化

- **血條顯示位置調整**：將血條的繪製移至UI類，確保不被其他元素遮蓋，特別是在暗色背景下。
- **規劃好的方法**：新增方法以檢查怪物是否在攝像機可見範圍內，提升程式的結構清晰度。

### ⚙️ 編碼優化

- **簡化代碼結構**：通過創建本地變數和循環加速檢查，加速運行效率並提升代碼可讀性。
- **條件檢查優化**：專注於只檢查怪物類，隨著代碼的修正，逐漸減少不必要的邏輯判斷。

### 📈 大型血條設計

- **製作大型血條框架**：為boss設計一個更明顯的血條框架，強調戰鬥的獨特性及重要性，提升玩家的遊戲體驗。

### 🎬 新增過場動畫

- **開始創建過場動畫**：計劃在未來的開發過程中將過場動畫整合進遊戲，以增強故事情節的表現。
